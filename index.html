Island Hopper

Island Hopper is a travel and tourism management system designed for local travelers, international tourists, families, backpackers, and travel agencies.

The platform provides detailed destination information, flexible itinerary planning, budget estimation, travel navigation, real-time updates, and convenient booking options.

The system helps boost local tourism by making destinations more accessible and appealing, supporting community economic growth, streamlining bookings, and enhancing the Philippines' reputation as a top travel destination.

Technology Stack

  • Programming Language: C# (ASP.NET Core MVC)
  • Database: SQL Server Management Studio (SSMS)
  • APIs: OpenStreetMap API, Geolocation API, PayPal REST API

ASP.NET Core MVC provides a flexible framework for developing a modern web application aligned with the project's requirements.

SSMS ensures reliable data storage and management, while APIs enhance the platform with live data integration, scalability, security, and efficient performance.
