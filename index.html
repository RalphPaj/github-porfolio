Island Hopper PH

Travel & tourism system for itinerary planning, booking, and destination discovery.

Island Hopper PH is a travel and tourism web platform built for local travelers, international tourists, families, backpackers, and travel agencies. It delivers detailed destination information, flexible itinerary planning, budget estimation, and convenient booking options in one system. The platform also supports travel navigation, real-time updates, and personalized trip customization to match different travel styles.

Beyond individual users, the system helps promote local tourism by making destinations more accessible and attractive, supports community-level economic growth, streamlines booking operations for organizations, and strengthens the Philippines' reputation as a top travel destination.

Tech Stack:
C# (ASP.NET Core MVC), SQL Server Management Studio (SSMS), OpenStreetMap API, Geolocation API, PayPal REST API.

BreezeEase

Mobile AC service system with predictive maintenance and smart recommendations.

BreezeEase is a mobile AC service application that helps customers book and track services, share their location for pickup and delivery, and receive maintenance reminders and status updates. It also analyzes AC unit and checklist data to estimate breakdown risk and generate smart, practical maintenance recommendations.

Tech Stack:

Royo Dry Goods Store

Automated sales and inventory management system with real-time tracking, role-based access, and secure data backup.

Royo Dry Goods Store is a comprehensive sales and inventory management system designed to streamline retail operations. The system automates critical business processes including sales recording, inventory monitoring, and transaction management, enabling staff to work more efficiently while maintaining accuracy.

Key features include automated sales transaction processing for cashiers and managers, real-time inventory tracking that updates automatically upon sales or stock arrivals, and a centralized database for secure storage of sales records. The system prevents stockouts by identifying low-stock items and facilitating data-driven restocking decisions. Additionally, it includes audit logs for tracking user activities and expenses, role-based access control for different staff levels (cashier, assistant manager, owner/manager), and automatic database backups on a paid hosting service to ensure data security and recovery.

Tech Stack:
Database: SQL Server, Hosting: Paid hosting service with automated backups, Security: Role-based access control, audit logging.

Chill & Swirl

Ice cream shop POS system with order management, inventory tracking, and streamlined sales processing.

Chill & Swirl is a custom Point of Sale system designed specifically for ice cream shops to streamline their operations. The system provides an intuitive and user-friendly interface that simplifies order processing, reduces transaction time, and enhances customer experience. It enables efficient tracking of inventory levels, ensures accurate sales recording, and facilitates smooth payment processing.

With Chill & Swirl, ice cream shop staff can quickly process customer orders, manage product availability in real-time, and generate sales reports to support business growth. The system is built to minimize operational overhead while maintaining accuracy in both sales and inventory management, making it an ideal solution for ice cream retail operations.

Tech Stack:
Point of Sale System, Inventory Management, Sales Processing, Real-time Data Tracking.
